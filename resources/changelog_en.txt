next release (current development snapshot):
  - Added experimental support for J2534 interfaces
  - Added experimental support for AT command controlled interfaces
  - Added experimental support for SSM2 over ISO-15765 (CAN)
  - Added support for the old SSM1 communication protocol
  - Added basic support for additional control units: ABS/VDC, CC, A/C
    (SSM1 only, definitions incomplete)
  - Added support for further engine and transmission control units 
    (OBD2 SSM1 controllers and additional SSM2 control unit addresses)
  - Added support for a program variant with a small user interface, which
    better fits to very small screens (e.g. used in car computers etc.)
  - Added support for automated program startup using command line parameters
  - Added Turkish translation
  - Lots of internal changes, code rework, modularization
  - Fixed some minor bugs
